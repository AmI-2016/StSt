AmI Steps and Features

AmI Main Steps

Sensing

Thanks to some sensors, the system understands the posture of the user and the brightness of the room.

Reasoning

The calculation of the difference between the actual posture and the correct one; the comparison between the room brightness and a reference brightness.

Acting

A vibration advises the user if he assumes a wrong posture; a sound signal and a display message warn the user about room brightness.

Interacting

The user can review the data collected by the system during the day, month, year.

AmI Features

Sensitive

The system is able to recognize a wrong posture and room brightness.

Responsive

The system is able to alert the user.

Adaptive

The system is able to adapt itself to every kind of body and it will start only when it is required.

Transparent

The system intervenes only when the user needs help.

Ubiquitous

The system is based on a wearable device. The user can install it in any environment he thinks to be good for studying!

Intelligent

The system is able to elaborate data and to understand when the user has to be warned.
