Vision

Study Station is a system thought as a wearable device that provide to avoid some health issues caused by wrong postures. The benefit is to leave the body relaxed even after long periods of sitting. The users interested could be all those people who spend a lot of time sitting. Students are supposed to stay a lot of time on desks during class lectures; a wrong posture may cause backache and also body injury if maintained over time. Our system may prevent it. People who already have those kind of habits and problems may get a useful help from the use of our device!

The device helps to prevent and correct the wrong posture through a signal used to warn users as, for example, a vibration from the device itself or a displayed message on the personal computer and smartphones. Furthermore, it gives some suggestions to modify their own workstation. It collects some statistics through sensors placed on some specific and critic points of the body (for example neck, shoulders, lumbar spine) during the session.

AmI Main Steps

AmI Step Description
Sensing Through some sensors our ambient understand the posture of the user.
Reasoning The calculation of the difference between the actual posture and the correct one.
Acting A signal as a vibration and/or a light advises the user of the wrong posture.
Interacting First the user may start the analysis whenever it's not active yet. Eventually the user can review the data collected by the ambient during the day, month, year.

AmI Features

AmI feature Description
Sensitive The system is able to recognize a wrong posture.
Responsive It's able to alert the user in case of a wrong posture.
Adaptive The system is able to adapt itself to every kind of body and it will start only when needed.
Transparent Only when the user need a help the device intervenes. Otherwise neither him neither other people notice it.
Ubiquitous The system is based on a wearable device. You can bring it wherever you need it.
Intelligent It's able to elaborate data and to understand when alert users.
